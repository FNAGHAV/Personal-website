From January 2023 to June 2024, I participated in the Data Analytics and Visualization Boot Camp at the University of Pennsylvania. During this intensive program, I received comprehensive training in various analytical and visualization techniques. This website was developed to further enhance my skills and provide an interactive platform to showcase my achievements and capabilities in data analytics and visualization. Below are the projects successfully completed during the boot camp:

Project 1: Decade Box Office Trends

GitHub link

Summary: The research anticipates variability in movie genres' popularity from 1990 to 2019, with an overall revenue growth despite economic and social fluctuations.
Role: Worked in a team of four to perform data cleaning and processing, data analysis, visualization, and statistical analysis.
Tools: Pandas, Pathlib, Numpy, Matplotlib, Scipy, JSON, Requests



Project 2: Crowdfunding ETL Pipeline

GitHub link

Summary: The Crowdfunding Data Processing Project focused on building an ETL pipeline to extract and transform data from Excel files, create four CSV files, develop an Entity-Relationship Diagram (ERD), establish a table schema, and upload the CSV file data into a PostgreSQL database.
Role: Worked in a team of four to process crowdfunding data from Excel files and transform it into a structured database using Python and PostgreSQL.
Tools: Python, Pandas, PostgreSQL, Jupyter Notebook, Matplotlib, QuickDBD



Project 3: South Philadelphia Community Fridge

GitHub link

Summary: Utilizes food donation data to create visualizations and analysis, supporting community outreach and engagement.
Role: Collaborated on data cleaning and processing; developed a Choropleth map for the South Philadelphia Community Fridge, created grid spaces, and configured the analysis notebook for local Voila compatibility to facilitate interactive visualization.
Tools: Python, Pandas, ipywidgets, folium, HTML, Voila, matplotlib, plotly, datetime, calendar



Project 4: Heart Attack Analysis and Prediction Project

GitHub link

Summary: Utilized the 2022 CDC BRFSS survey dataset to develop machine learning models predicting heart attack risk, aiming to enhance healthcare resource allocation and reduce costs.
Role: Conducted all aspects of data analysis, machine learning model development, and website design independently after initial group collaboration on the dataset.
Tools: Spark, SQL, Python, Pandas, Matplotlib, Plotly, Folium, Logistic Regression, Random Forests, Gradient Boosting, XGBoost, TensorFlow, PCA, Adaptive Synthetic Sampling, HTML, CSS, JavaScript
